## AI-Powered Smart Gardening System

* Developed an AI-powered irrigation forecasting system using data processing, supervised machine learning, and explainable AI techniques.
* Designed to help gardeners make informed watering decisions, optimize water usage, and promote sustainable gardening practices.
* Evaluated and compared six machine learning models: XGBoost, Decision Tree, Naive Bayes, Support Vector Machine (SVM), K-Nearest Neighbors (KNN), and Random Forest.
* Applied thorough data normalization and hyperparameter tuning to improve model performance.
* XGBoost emerged as the best-performing model, achieving the highest Precision, Recall, F1-Score, and AUC, while maintaining the lowest Log Loss.
* Enabled accurate irrigation predictions along with reliable probability estimates for real-world gardening applications.
* Integrated SHAP (SHapley Additive Explanations) to provide transparency and interpretability of model predictions.
* Analyzed the impact of key features such as soil moisture, temperature, and time on irrigation decisions.
* Promoted efficient water management, healthier plant growth, and data-driven gardening through AI-powered decision support.
